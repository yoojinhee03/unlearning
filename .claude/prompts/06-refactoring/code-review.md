# 코드 리뷰 & 개선 제안

Review [FILE/FUNCTION] and suggest improvements for:
1. Readability and clarity
2. Performance (only if there's a real issue)
3. Error handling completeness
4. Edge cases not handled
5. Code duplication that could be extracted

For each suggestion: explain WHY it's better, not just what to change.
Don't refactor for its own sake — only flag real problems.
